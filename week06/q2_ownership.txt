=== 任務 1：探索 Linux 目錄結構 ===
執行以下指令：
ls /

bin   dev  home  lib    lost+found  mnt  proc  run   srv  tmp  var
boot  etc  init  lib64  media       opt  root  sbin  sys  usr

請從上面的結果中，說明以下 5 個目錄各是做什麼的：
/home：直接填入：
/etc：存放系統與應用程式的設定檔，例如 /etc/passwd 儲存帳號資料、
       /etc/group 儲存群組資料，是系統管理員最常操作的目錄。
/var：存放會隨時間變動的檔案，例如系統日誌（/var/log）、
       郵件佇列、資料庫資料等，var 代表 variable（可變的）。
/tmp：存放臨時檔案，系統重新開機後通常會自動清空，
       所有使用者都可以在此建立暫時使用的檔案。
/usr：存放系統安裝的程式、函式庫與說明文件，
       例如 /usr/bin 放一般指令，/usr/lib 放函式庫。
